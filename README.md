# Book APP
## A React Native App


## Installation

  The app is fully developed with the react-native-cli. 
  
 - Clone the project
 - Navigate to the root directory of the project and run <b>npm install</b>.
 - After installing npm packages run <b>react-native run-android</b> for ANDROID
